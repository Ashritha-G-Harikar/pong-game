# pong-game
This project will replicate the famous arcade game : Pong.

The rules are simple : play like badminton.
The controls for right paddle are : 
    a. Press ↑ to go up.
    b. Press ↓ to go down.

The controls for left paddle are :
    a. Press 'w' to go up.
    b. Press 's' to go down.

Enjoy!
